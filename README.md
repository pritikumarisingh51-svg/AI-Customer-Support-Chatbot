# AI Customer Support Chatbot 🤖

## 📌 Overview
This project is an AI-powered customer support chatbot developed using Python and Streamlit.

The chatbot interacts with users, answers common customer queries, and provides a simple conversational experience.

---

## 🚀 Features
- Interactive chatbot UI
- AI-like conversation
- Customer support responses
- Greeting detection
- Fast response system
- Streamlit web application

---

## 🛠️ Technologies Used
- Python
- Streamlit
- NLP Basics
- Random Module

---

## ▶️ How to Run

### Install Streamlit
pip install streamlit

### Run the application
python -m streamlit run app.py
